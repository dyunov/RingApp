How to cite this site

Was the Database of Ring Theory useful to you? Please consider acknowledging the resource so that others can find it as well!

To cite the data set, You can easily obtain an APA citation or BibTeX citation by visiting the data set GitHub repository and using the "Cite this repository" item visible in the "About" information. This is probably what you want if you are using information you found from the database.

There is a corresponding button for the website code itself in the website GitHub repository if you need that instead.

If you have a success story about how the site helped you in a publication or otherwise, we would also be appreciative of a notice via the suggestion form!
